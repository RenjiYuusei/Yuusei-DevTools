# Changelog

All notable changes to this project will be documented in this file.

## [1.1]
### Added
- Application tab for inspecting Local Storage, Session Storage, and Cookies.
- Ability to delete storage items and cookies.
- Text search filter in the Network tab toolbar.
- `CHANGELOG.md` file.

## [1.0]
### Added
- Tabbed interface for Network details pane (Headers, Payload, Preview, Response, Timing).
- Sources tab with hierarchical file tree support.
- Icons for different file types in Sources tree.
- cURL generation for network requests.
- Dark theme styling.

### Fixed
- Fixed issue where DevTools window closed when clicking outside on mobile (changed to normal window).
- Handled 'Already attached' debugger errors with auto-retry.
